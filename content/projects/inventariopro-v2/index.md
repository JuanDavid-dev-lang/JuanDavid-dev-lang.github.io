# InventarioPro v2.0 - Sistema Integral de Gestión Predictiva de Inventario

## Visión General

El proyecto **InventarioPro V2** consiste en el desarrollo de un sistema integral de gestión de inventario diseñado para administrar productos, controlar entradas y salidas de mercancía, generar reportes y facilitar la toma de decisiones mediante análisis de datos e inteligencia artificial.

---

## 🛠 Arquitectura del Sistema

La arquitectura de la plataforma combina diversas tecnologías de vanguardia para garantizar rendimiento y escalabilidad:

- **Frontend**: Desarrollado en **React** con componentes interactivos e indicadores visuales.
- **Backend API REST**: Construido en **PHP** estructurado para el control operacional de mercancías.
- **Base de Datos**: **MySQL** relacional para la persistencia transaccional de usuarios, productos y movimientos.
- **Módulos de Machine Learning**: Scripts en **Python** especializados en la predicción de demanda de stock.
- **Big Data Processing**: **Apache Spark** para el procesamiento analítico de grandes volúmenes de datos históricos.
- **Despliegue & Orquestación**: Entorno completamente contenedorizado utilizando **Docker**.

---

## 🚀 Funcionalidades Clave

- **Paneles de Indicadores (KPIs)**: Monitoreo en tiempo real de rotación e inventario.
- **Alertas de Stock Bajo**: Notificaciones automáticas basadas en umbrales de reabastecimiento.
- **Predicción de Demanda por IA**: Algoritmos que estiman compras futuras para evitar sobrestock.
- **Importación Masiva de Datos**: Módulo para carga rápida de datos desde archivos Excel.
- **Autenticación y Seguridad**: Control de acceso basado en roles (RBAC).
